## Sortable tables in elm-mdl
With much inspiration from    https://github.com/evancz/elm-sortable-table

To Demo, run this
```sh
git clone https://github.com/cmditch/elm-mdl-sortable-table.git
cd elm-mdl-sortable-table
elm-reactor
```
